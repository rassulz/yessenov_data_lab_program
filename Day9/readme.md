Overview
Во время сна мозг и тело проходят через несколько стадий. В лаборатории сна их записывают сразу несколькими каналами: электроэнцефалограмма (EEG), мышечный тонус (EMG), движения глаз (EOG), пульс и дыхание. Специалист размечает запись по 30-секундным эпохам (epoch), приписывая каждой одну из стадий.

Эту разметку давно пытаются автоматизировать. Задача трудная: границы между стадиями размыты, и даже опытные сомнологи нередко расходятся в оценке одной и той же эпохи. Идеальная точность здесь недостижима в принципе.

Ваша задача: по показаниям датчиков предсказать стадию сна для каждой эпохи в тестовой выборке. Соревнование идёт по метрике macro-F1.

Данные синтетические и созданы для учебных целей. Это не клинический инструмент.

Evaluation
Метрика: macro-F1 (макро-усреднённая F-мера по 4 классам). Все классы важны одинаково независимо от их доли в данных.

Формат сабмита: CSV с заголовком и по строке на каждую эпоху из test.csv:

id,sleep_stage
9000,1
9001,3
9002,0
Публичный лидерборд (public) считается по 30% тестовых строк и виден во время соревнования. Закрытый лидерборд (private) считается по остальным 70% и открывается в конце. Итоговое место определяет закрытый лидерборд, так что не подгоняйте решение под публичную часть.

Как определяется победитель
Основное место: по macro-F1 на закрытом лидерборде.
Отдельно отмечается лучший подход: за ясное, обоснованное и воспроизводимое решение, оценивается по защите. Высокое место в лидерборде и сильный подход это не всегда одно и то же.

Правила честной игры
Только предоставленные данные. Внешние датасеты и ручная разметка теста запрещены.
Лимит сабмитов: до 20 в день.
Код воспроизводим. Решение, которое нельзя повторить, на защите не засчитывается.
Итоговое место определяет закрытый лидерборд, а не публичный.


Dataset Description
Целевая переменная sleep_stage:

0 = Wake (бодрствование)
1 = Light (поверхностный сон, N1 и N2)
2 = Deep (глубокий сон, N3)
3 = REM (фаза быстрого сна)
Основные признаки (нормализованные значения с каналов записи):

eeg_delta_power, eeg_theta_power, eeg_alpha_power, eeg_sigma_power, eeg_beta_power, eeg_gamma_power, eeg_slow_osc_power, eeg_spectral_entropy, eeg_spindle_density, eeg_kcomplex_rate, emg_chin_tone, emg_tone_variance, eog_movement_density, eog_amplitude, heart_rate_mean, heart_rate_variability, respiration_rate, respiration_variability, spo2_mean, body_movement_index.

Отдельный признак:

eog_burst_index: индекс всплесков движений глаз с отдельного канала EOG. Этот канал работал не на всех записях, поэтому значение есть примерно для половины эпох, а у остальных пропущено (пустая ячейка). Как с этим поступить, решаете сами.
Файлы:

train.csv: 9000 эпох. Столбцы id, признаки, sleep_stage.
test.csv: 5000 эпох. Столбцы id и признаки. Стадию нужно предсказать.
sample_submission.csv: пример правильного формата сабмита.
